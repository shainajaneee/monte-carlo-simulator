The Algorithm

This project implements a Monte Carlo Simulation—a mathematical technique used to estimate the possible outcomes of an uncertain event.

O(n)

Probabilistic Iteration: We execute $S$ simulations over $T$ trades, generating a random variable $X$ to determine win/loss/breakeven states.

Σ

Data Aggregation: Every result is sorted to find the Median (P50), providing a statistically grounded "Most Likely" outcome.
